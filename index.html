    Sobre o Brooks

    O Brooks é um buscador inteligente de livros que integra a Google Books API e a API Gemini para oferecer recomendações personalizadas de leitura.

    O projeto foi desenvolvido com o objetivo de demonstrar a integração entre APIs utilizando JavaScript, HTML, CSS e Bootstrap. Além de apresentar livros de diferentes gêneros, a aplicação gera um resumo e indica o perfil de leitor recomendado, apresentar os principais temas abordados e fornecer uma curiosidade sobre a obra.

    O principal propósito do Brooks é incentivar a descoberta de antigos e novos livros de forma intuitiva e acessível para qualquer leitor.
